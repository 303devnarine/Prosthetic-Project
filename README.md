# Arduino Prosthetic Project
## by Dev Narine

### Why design a prosthetic?
I wanted to get into the eyes of a biomedical engineer in the prosthesis field and learn the work it takes to design, prototype, and essentially build a prostheic limb.

### What Arduino boards does this project use?
I used the Arduino Uno R4 WiFi as the part that controls the movement, while I used the Arduino Nano RP2040 as the central brain that communications between the control and the actual limb itself.

### Which board runs which .ino file?
"prostheticProjControl.ino" is ran on the Arduino Uno R4 WiFi, while "prostheticBLECentral.ino" is ran on the Arduino Nano RP2040.

### What components were used for this project, other than the main boards?
- 1 16*2 LCD screen (with I2C wire connections)
- 1 4 AA Battery Pack
- 5 9g Servo Motors
- Joystick Module
- Many Wires
